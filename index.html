Introduction

In this proposal we describe our project of creating an interactive web application to remotely control a TurtleBot through the web browser instead of command line. We aim to achieve three goals through our project. First, we will create a web application which will allow the user to use keyboard or click button on the website to remotely operate a TurtleBot to complete basic tasks including move forward, move backward, turn left, and turn right. Second, through the web application, users will be able to use speech commands to control a TurtleBot to complete the same tasks. Third, our web application will display the camera view of the TurtleBot which is being tele-operated in the web browser. Therefore, users can see what the TurtleBot sees in the real time world. Compared with traditional lab tour, our projeect might provide young students a more intuitive and interactive way to learn about the TurtleBots.
